# PBL1_Projekt

Niniejsza dokumentacja dotyczy projektu realizowanego na przedmiocie PBL w semestrze 2025Z. Celem projektu było stworzenie urządzenia IOT które odpowiadałoby tematyce przewodniej „Opieka nad zwierzętami”. Postanowiliśmy opracować urządzenie monitorujące warunki w hodowli drobiu pod kątem zgodności z obowiązującymi normami prawnymi. Do tego stworzyliśmy prototyp urządzenia z wykorzystaniem mikrokontrolera ESP32. 
Nasz system składa się z aplikacji mobilnej, strony internetowej, urządzeń zawierających zestaw czujników i z korzenia sieci mesh oraz urządzenia zawierającego wagę i czytnik NFC.
